# Client Onboarding Checklist Template

This checklist is automatically added to every new Trello client card.

━━━━━━━━━━━━━━━

## Default Checklist

```text
Send contract
Create invoice draft
Schedule kickoff call
Request brand assets
Confirm project scope
```

━━━━━━━━━━━━━━━

## Optional Extra Checklist Items

You can customize the workflow and add more checklist items, such as:

```text
Create client portal
Send contract for signature
Create first invoice
Collect brand guidelines
Collect website access
Collect social media access
Schedule discovery call
Confirm project timeline
Assign internal project owner
Prepare kickoff agenda
```

━━━━━━━━━━━━━━━

## Usage

Use the default checklist for a simple agency onboarding process.

Use the optional checklist items if you want to make the workflow more detailed for larger client projects.
